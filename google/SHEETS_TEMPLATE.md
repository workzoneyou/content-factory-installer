# Google-таблица пайплайна

Пайплайн управляется одной Google-таблицей с несколькими листами. n8n читает из неё
темы, промпты, офферы и пишет бэкап.

## Как получить

**Проще всего** — попросить копию рабочего шаблона (File → «Создать копию»),
затем очистить клиентские данные (см. ниже). Если делаете с нуля — создайте таблицу
с листами и колонками ниже.

После копирования: **File → Share** дайте доступ Google-аккаунту, под которым будете
делать OAuth в n8n. ID таблицы — из URL (`.../spreadsheets/d/`**`ЭТОТ_ID`**`/edit`) —
впишите в `.env` → `GOOGLE_SHEET_ID`, затем `bash n8n/setup.sh`.

## Листы

| Лист | Назначение | Ключевые колонки |
|---|---|---|
| **КОНВЕЙЕР** | Очередь тем на генерацию | `Главный ключ`, `Статус` (пусто/`новая`→берётся, `в процессе`/`Ошибка`→нет), `Город` |
| **ПРОМПТЫ** | Системники агентов | `Имя агента` \| `Системный промпт` (строки: Писатель, Арт, Архитектор, Журналист, Исследователь, Саппорт…) |
| **НАСТРОЙКИ** | Параметры (ключ-значение) | напр. `img_url` (референс для картинок), `wp_url`, язык |
| **ОФФЕРЫ** | База офферов для вставок | `id`, `Название`, `Ссылка`, `Текст`, тип (мягкий/жёсткий) |
| **РУБРИКИ** | Рубрики (категории) | список рубрик |
| **ОПУБЛИКОВАНО** | База для внутренней перелинковки | `ID`, `Ссылка`, `Тема` |
| **БЭКАП СТАТЕЙ** | Бэкап (пишется автоматически) | `ID` \| `Дата/время` \| `Тема` \| `Текст` \| `Картинки` |

## ⚠️ Обязательно при новом клиенте

- **Лист ОПУБЛИКОВАНО — ОЧИСТИТЬ (оставить только заголовки).** Если там останутся
  чужие/старые URL — пайплайн проставит внутренние ссылки на несуществующие страницы (404).
  Он наполняется сам по мере публикации реальных статей.
- **ОФФЕРЫ** — заменить на реальные офферы клиента (ссылки/тексты).
- **ПРОМПТЫ** — при желании подстроить голос под клиента (особенно «Писатель» и «Арт»).
- **КОНВЕЙЕР** — залить темы клиента.
- **БЭКАП СТАТЕЙ** — оставить пустым (заполнится сам).

## OAuth в n8n

n8n → **Credentials → New → Google Sheets OAuth2 API**:
1. В Google Cloud Console создайте OAuth-клиент (тип Web), redirect URI —
   `https://<N8N_DOMAIN>/rest/oauth2-credential/callback`.
2. Client ID / Secret → в креденшел n8n, пройдите «Connect».
3. Тем же аккаунтом (или отдельным) настройте Google Drive OAuth для бэкапа статей
   (UpdraftPlus на сайте + при желании нода Drive в n8n).
